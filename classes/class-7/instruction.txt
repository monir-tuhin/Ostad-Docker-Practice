=> ADD ./package.json .  ||   COPY ./package.json : These two uses for copying files and folders. But should I use ADD or COPY ? Both work for copy but ADD has extra features beside copying . Here are details below :
    - ADD :
        > ADD http://aaaa.com : Download from link or url
        > ADD https://github.com/monir-tuhin/Ostad-Docker-Practice.git : Do checkout git repository and add repositories file
        > Muloto add fetch the files but copy does not fetch the file
    - COPY:
        > COPY ./package.json . : Only copy the files
        > Not fetching any files from any link or url
        > Main task of copy is only copying the file
        > For copying a file or folder we should use only COPY because it is slight faster than ADD
